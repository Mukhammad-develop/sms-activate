# Политика конфиденциальности

**Последнее обновление: 1 ноября 2025**

## 1. Введение

Настоящая Политика конфиденциальности описывает, как мы собираем, используем и защищаем вашу информацию при использовании нашего Telegram-бота SMS-Activate (далее - "Сервис"). Используя Сервис, вы соглашаетесь со сбором и использованием информации в соответствии с настоящей политикой.

## 2. Информация, которую мы собираем

### 2.1 Персональная информация
- **Telegram User ID**: Ваш уникальный идентификатор Telegram (числовой)
- **Telegram Username**: Ваш публичный username в Telegram (если доступен)
- **Имя**: Ваше отображаемое имя в Telegram
- **Предпочтения языка**: Язык, выбранный для интерфейса бота

### 2.2 Информация о транзакциях
- **История покупок**: Купленные услуги, полученные номера телефонов, ID заказов
- **Информация о балансе**: Баланс счета и записи транзакций
- **Платежные данные**: Суммы транзакций, временные метки, описания
- **Детали заказов**: ID активации, номера телефонов, коды сервисов, страны

### 2.3 Техническая информация
- **Сообщения**: Команды и взаимодействия с ботом
- **IP-адрес**: Может быть собран через API Telegram
- **Данные об использовании**: Частота использования сервиса, доступные функции

### 2.4 Содержимое SMS
- **SMS-коды**: Коды верификации, полученные через наш сервис
- **Текст SMS**: Полный текст полученных SMS-сообщений
- **Номера телефонов**: Временные виртуальные номера телефонов

## 3. Как мы используем вашу информацию

### 3.1 Предоставление услуг
- Обработка и выполнение ваших запросов на услуги
- Предоставление виртуальных номеров телефонов для верификации
- Доставка SMS-сообщений вам
- Управление балансом счета и транзакциями
- Отправка подтверждений заказов и уведомлений

### 3.2 Управление аккаунтом
- Создание и ведение вашего пользовательского аккаунта
- Аутентификация вашей личности
- Обработка возвратов и отмен
- Ведение истории транзакций

### 3.3 Коммуникация
- Ответы на ваши запросы и запросы поддержки
- Отправка обновлений сервиса и уведомлений
- Доставка обновлений статуса заказа

### 3.4 Соответствие законодательству
- Соблюдение применимых законов и правил
- Ответы на юридические запросы и судебные приказы
- Защита наших прав и предотвращение мошенничества

### 3.5 Улучшение сервиса
- Анализ шаблонов использования (анонимизированный)
- Улучшение функциональности сервиса
- Устранение технических проблем

## 4. Хранение данных и безопасность

### 4.1 Место хранения
- Данные хранятся на защищенных серверах
- Данные пользователей Telegram хранятся в соответствии с политикой Telegram
- Данные транзакций зашифрованы при хранении

### 4.2 Меры безопасности
- Шифрование SSL/TLS для передачи данных
- Безопасные механизмы аутентификации
- Регулярные проверки безопасности
- Контроль доступа и мониторинг

### 4.3 Хранение данных
- **Активные аккаунты**: Данные хранятся, пока аккаунт активен
- **Неактивные аккаунты**: Данные хранятся в течение 2 лет после последней активности
- **Записи транзакций**: Хранятся в течение 7 лет для юридических/соответствия целей
- **Содержимое SMS**: Удаляется через 30 дней или по запросу пользователя

## 5. Сторонние сервисы

### 5.1 SMS-Activate API
- Мы используем SMS-Activate API для предоставления виртуальных номеров телефонов
- Ваши запросы передаются в SMS-Activate
- Политика конфиденциальности SMS-Activate применяется к их сервису
- Мы не контролируем практики обработки данных SMS-Activate

### 5.2 Telegram
- Сервис работает через платформу Telegram
- Политика конфиденциальности Telegram применяется: https://telegram.org/privacy
- Мы получаем данные пользователей через Telegram Bot API
- Мы не храним ваш пароль Telegram или учетные данные для входа

### 5.3 Платежные процессоры
- Обработка платежей осуществляется администратором (вручную)
- Сторонние платежные процессоры не используются
- Все платежные коммуникации являются частными и безопасными

## 6. Обмен данными и раскрытие информации

### 6.1 Мы НЕ продаем ваши данные
- Мы никогда не продаем, не сдаем в аренду и не обмениваем вашу персональную информацию
- Ваши данные не передаются рекламодателям или маркетологам

### 6.2 Ограниченный обмен
Мы можем делиться информацией только в этих обстоятельствах:
- **С SMS-Activate**: Необходимо для выполнения запросов на услуги
- **Юридические требования**: Когда требуется по закону или судебному приказу
- **Бизнес-передачи**: В случае слияния, поглощения или продажи
- **С согласия**: Когда вы явно авторизуете обмен

### 6.3 Анонимизированные данные
- Мы можем использовать анонимизированные, агрегированные данные для аналитики
- Эти данные не могут идентифицировать отдельных пользователей

## 7. Ваши права и выбор

### 7.1 Доступ к вашим данным
- Просмотр баланса счета через `/balance`
- Просмотр истории транзакций через `/history`
- Проверка ваших заказов через `/myorders`

### 7.2 Удаление данных
- Запросите удаление аккаунта, связавшись с администратором
- Все персональные данные будут удалены в течение 30 дней
- Записи транзакций могут быть сохранены для юридического соответствия

### 7.3 Исправление данных
- Обновите предпочтения языка через `/language`
- Свяжитесь с администратором для исправления информации аккаунта

### 7.4 Отказ
- Прекратите использование Сервиса в любое время
- Отключите уведомления через настройки Telegram

## 8. Конфиденциальность детей

Наш Сервис не предназначен для пользователей младше 18 лет. Мы сознательно не собираем персональную информацию от детей. Если вы являетесь родителем или опекуном и считаете, что ваш ребенок предоставил нам персональную информацию, пожалуйста, свяжитесь с нами немедленно.
